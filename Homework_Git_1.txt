Задание 1.
1. Нужно создать в своём аккаунте на github.com пустой репозиторий, добавив файл Readme. Скопировать ссылка-SSH
2. Клонируем на локальный диск в компьютере этот репозиторий командой git clone ссылка-SSH.
Задание 2.
1. В склонированной папке репозитория на локальном диске создаём (копируем) необходимые файлы, или проект.
2. В терминале находясь в папке с клонированным репозиторием последовательно выполняем команды:
	-  git add . - добавление всех файлов (папок) к commit (можно указать конкретное имя файла)
	- git commit -m "Message" - зафиксировать изменения, с коментарием
	- git push   -  залить файлы с изменениями на github 

Задание 3.
1. Для получения (загрузки) изменений из github.com на локальный компьютер (если репозиторий уже ранее склонирован) необходимо перейти в папку с репозиторием и выполнить команду git pull.

Задание 4.
git status  -  показывает неотслеживаемые файлы
git log  -  показывает список комитов, когда были изменения
git remote -v  -  проверка связи репозитория с github (удалённая ссылка на данный репозиторий)
